# About this project
This is a very simple chrome extension that uses tinyURL api to generate a tinyurl that can be copied by a simple button click
### Languages used

# How to use and build
1) clone this repository
```
git clone https://github.com/256kib/generic-url-shortner.git
```
2) enter the project folder
```
cd generic-url-shortner
```
3) install required npm packages and build
```
npm install
npm run build
```
4) [load the build folder as unpacked extension](https://developer.chrome.com/docs/extensions/get-started/tutorial/hello-world?#load-unpacked)
